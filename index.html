Federated learning data poisoning attack

project overview

Federated Learning is a decentralized approach to machine learning that allows multiple devices or clients to collaboratively train a shared model without sharing their raw data. While FL offers privacy benefits, it introduces new vulnerabilities—one of the most significant being data poisoning attacks. In a data poisoning attack, malicious clients inject incorrect or manipulated data into the learning process, disrupting the global model's performance.This project explores the threat of data poisoning in federated learning, investigates potential attack strategies, and proposes defensive techniques to mitigate such risks.

Technologies Used

For writing the code to simulate federated learning and implement attacks.

For data visualization and impact analysis.
